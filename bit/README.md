# Bhojpur C++ - Information Theory Algorithms

In this folder, we maintain a collection of different algorithms applied in
various Information Theory applications.

## Source Code Compilation

```bash
$ make all
```

## Folder Cleanup

To clean all temporary files folder

```bash
$ make clean
```

To clean only pre-compiled objects folder

```bash
$ make clean-objs
```

To clean only pre-compiled library folder

```bash
$ make clean-libs
```
